# Productivity Tracker

## Project Structure

- `app/` - Main application code (Streamlit app, models, user management)
- `data/` - Data files (tasks, logs, users)
- `README.md` - Project overview and instructions

## Usage
Run the Streamlit app with:

```
streamlit run app/streamlit_app.py
```

## Folders
- `app/streamlit_app.py` - Main Streamlit app
- `app/models.py` - ProductivityEntry and related classes
- `app/user.py` - User management
- `data/tasks.txt` - Task categories
- `data/productivity_log.txt` - Productivity log
- `data/users.json` - User data
